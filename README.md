# 📊 Blinkit Sales Data Analysis using Python

## 🔍 Project Overview
This project performs end-to-end data analysis on Blinkit sales data using Python. The goal is to clean the dataset, calculate key performance metrics (KPIs), and generate insights using data visualization.

## 🎯 Business Objectives
- Analyze total and average sales performance
- Understand product category contribution
- Evaluate outlet performance across different factors
- Identify trends in sales and customer preferences

## 📌 KPIs Analyzed
- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Ratings

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
  
## 🔧 Data Cleaning
- Standardized inconsistent values in `Item Fat Content`
- Replaced values like 'LF', 'low fat' with 'Low Fat'
- Checked dataset structure using `.info()`, `.shape()`

## 📊 Analysis & Visualizations

### 1. Sales by Fat Content
- Pie chart showing contribution of Low Fat vs Regular items

### 2. Sales by Item Type
- Bar chart showing top-performing product categories

### 3. Fat Content by Outlet Location
- Stacked bar chart comparing sales across outlet tiers

### 4. Sales by Outlet Establishment Year
- Line chart showing sales trend over time

### 5. Sales by Outlet Size
- Pie chart showing distribution across outlet sizes

### 6. Sales by Outlet Location
- Horizontal bar chart showing location-based performance

## 📈 Key Insights
- low-fat products generated higher sales compared to regular products  
- Certain item categories contribute significantly more revenue  
- Tier 3 outlet locations showed higher overall sales  
- Medium-sized outlets performed better than small and high outlets  
- Sales trends vary based on outlet establishment year
